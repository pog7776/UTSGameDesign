# UTSGameDesign
Repo for Game Design digital game
<br>
<br>
<hr>

W / Space:        Jump

A / left arrow:   Left

D / right arrow:  Right

S / Down:         Crouch

Lmb / Ctrl:       Fire projectile

Rmb:              Slash attack

Esc:              Restart scene
